# Hi Dev Tools (Nuxt 3)

Bộ tool nhỏ phục vụ Dev:

- Request API Online
- JSON Formatter
- HTML Viewer
- Color CSS
- JWT Decoder
- MD5 Generator

## Cách chạy

```bash
npm install
npm run dev
```

Mặc định có SSR. API mock nằm ở `server/api/home.get.ts`.  
Biến môi trường public: `NUXT_PUBLIC_API_BASE` (mặc định `https://api.hifpt.vn`).
